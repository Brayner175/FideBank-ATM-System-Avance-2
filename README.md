# FideBank-ATM-System-Avance-2
Este proyecto consiste en el diseño e implementación de un sistema de Cajero Automático (ATM) desarrollado en Java, como parte del curso Programación Cliente Servidor Concurrente.

El sistema simula el funcionamiento de un cajero automático para la empresa FideBank, permitiendo a los clientes realizar diferentes operaciones bancarias de manera segura mediante autenticación con PIN.
Entre las operaciones disponibles se encuentran:

Apertura de cuentas, autenticación de usuarios mediante PIN, consulta de saldo, retiro de dinero, depósito de dinero, transferencia entre cuentas, generación de comprobantes de transacciones

El sistema está desarrollado utilizando programación orientada a objetos, integrando conceptos como clases, herencia, polimorfismo, manejo de excepciones, colecciones y una interfaz gráfica de usuario.
